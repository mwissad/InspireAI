# Inspire AI v4.6 — Data Strategy Copilot

> Turn your Unity Catalog into an actionable analytics strategy — powered by Databricks Foundation Models.

## Quick Deploy

```
Databricks > Compute > Apps > Create App > Custom App
Git URL: https://github.com/mwissad/InspireApp.git
Branch: main  |  Name: inspire-ai  |  Click Create
```

The Setup Wizard handles the rest.
